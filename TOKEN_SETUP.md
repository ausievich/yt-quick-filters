# Настройка токена для YouTrack API

## Проблема
Расширение не может получить доступ к YouTrack API без токена аутентификации.

## Решение

### Вариант 1: Получить Permanent Token (Рекомендуется)

Согласно [официальной документации YouTrack](https://www.jetbrains.com/help/youtrack/devportal/Manage-Permanent-Token.html), выполни следующие шаги:

1. **Войди в YouTrack** и нажми на свой аватар в правом верхнем углу
2. **Выбери "Profile"** из меню
3. **Перейди на вкладку "Account Security"**
4. **Нажми "New token"**
5. **Заполни форму**:
   - **Name**: "YT Quick Filters" (или любое другое название)
   - **Scope**: Выбери "YouTrack" (для доступа к issues, agile boards и т.д.)
6. **Нажми "Create token"**
7. **Скопируй токен** (он будет показан только один раз!)
8. **Добавь токен в браузер**:
   ```javascript
   // Выполни в консоли браузера на странице YouTrack:
   localStorage.setItem('youtrack_permanent_token', 'perm:твой_токен_здесь');
   ```

**Важно**: Токен имеет формат `perm:xxxxx.xxxxx.xxxxx` и показывается только один раз!

### Вариант 2: Перехват существующих запросов

Расширение автоматически пытается перехватить токены из существующих API запросов YouTrack.

### Вариант 3: Ручная настройка

Если у тебя уже есть токен, добавь его в localStorage:

```javascript
// В консоли браузера:
localStorage.setItem('youtrack_token', 'твой_токен_здесь');
```

## Проверка

После настройки токена:
1. Обнови страницу YouTrack
2. Открой консоль браузера
3. Должно появиться: `🔑 Found auth token: eyJhbGciOiJSUzI1NiIs...`
4. На карточках задач должны появиться оранжевые индикаторы с днями

## Безопасность

- Токены хранятся только в localStorage браузера
- Никогда не передавай токены третьим лицам
- Регулярно обновляй токены для безопасности
